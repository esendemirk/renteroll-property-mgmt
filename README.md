# Renteroll - Property Management Dashboard

A minimal, luxurious property management dashboard built with HTML, CSS, and vanilla JavaScript.

## Features

- **Login Page** - Clean authentication interface
- **Dashboard** - Overview of properties, tenants, and maintenance requests
- **Properties** - Manage properties with expandable tenant details
- **Residents** - View all tenants with contact information
- **Payments** - Process rent payments and schedules
- **Forms** - Download rent increase and tenant moveout forms

## Design

- Color Palette: Pistachio Green (#93C572)
- Typography: Plus Jakarta Sans
- Fully responsive and mobile-friendly
- Minimal, modern, luxe aesthetic

## Tech Stack

- Pure HTML/CSS/JavaScript
- No build process required
- No dependencies
- Hosted on Cloudflare Pages

## Deployment

This project is designed to be deployed on Cloudflare Pages with zero configuration.

## Pages

- `index.html` - Login page (landing page)
- `dashboard.html` - Main dashboard
- `properties.html` - Property management
- `residents.html` - Tenant directory
- `payments.html` - Payment processing
- `forms.html` - Document downloads
